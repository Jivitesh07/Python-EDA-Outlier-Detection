# Python-EDA-Outlier-Detection

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a house price dataset using Python.
It focuses on understanding data distribution, detecting outliers using the IQR method, and preparing a clean dataset for further analysis.

## Tools & Technologies
1. Python
2. Pandas
3. NumPy
4. Matplotlib

## Key Steps Performed
1. Data loading and inspection
2. Descriptive statistics
3. Distribution analysis (histogram & boxplot)
4. Outlier detection using IQR
5. Outlier handling using capping
6. Correlation analysis

## Outcome
1. Identified and treated outliers effectively
2. Improved data quality
3. Generated a clean dataset ready for modeling
